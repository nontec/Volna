# Volna

(ENG)
"Volna" turns your smartphone into a tactical digital walkie-talkie. The core concept is based on the "press to talk, release to listen" principle. The interface is fully optimized for blind or gloved operation thanks to a massive PTT button that occupies the lower half of the screen and changes color depending on the channel status.
The app is privacy-focused, designed for minimal battery consumption, and built to operate in unstable networks.

(РУС)
Приложение «Volna» превращает смартфон в тактическую цифровую рацию. В основе концепции лежит принцип «зажал — говори, отпустил — слушай». Интерфейс полностью оптимизирован под управление вслепую или в перчатках за счет огромной PTT-кнопки, занимающей нижнюю половину экрана и меняющей цвет в зависимости от состояния эфира.
Приложение ориентировано на приватность, минимальный расход батареи и работу в нестабильных сетях.
